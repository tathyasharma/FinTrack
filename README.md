# FinTrack 📈

A React Native crypto portfolio tracker app built with Expo.

## Features
- 🔴 Live crypto prices in INR via CoinGecko API
- ⭐ Personal watchlist to track favourite coins
- 💼 Portfolio tracker with real-time value calculation
- 🔄 Pull to refresh live market data

## Tech Stack
- React Native (Expo)
- React Navigation (Bottom Tabs)
- CoinGecko Public API
